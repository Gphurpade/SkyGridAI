# SkyGridAI
SkyGridAI is a data processing and prediction project for weather and environmental data. It includes scripts for parsing grid data, preprocessing, merging datasets, training models, and generating predictions.

## Project Structure
- app.py: Main application entry point
- scripts/: Processing, training, visualization, and mapping utilities
- data/: Input weather and rainfall grid files
- output/: Generated CSV files and heatmaps
- models/: Trained model artifacts
- tests/: Prediction-related tests
